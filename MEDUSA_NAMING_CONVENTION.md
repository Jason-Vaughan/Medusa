# Medusa Naming Convention - "Mythology Meets Code"

## Philosophy
*Why use boring technical names when you can use names of powerful and legendary figures who got things done? Medusa embraces personality while building revolutionary tech.*

---

## Current Names

### Core System: "MEDUSA" - The Multi-Headed Coordinator
- **Medusa Server:** MCP server handling multiple workspaces (like Medusa's snake hair)
- **Medusa Chat Protocol:** The communication protocol between workspaces
- **Medusa Dashboard:** Web interface that monitors everything

### Scripts & Tools: "The Coven" - Creatures of Automation
- **ZombieDust.js** *(formerly ai-auto-loop-hack.js)*: Transforms AIs into autonomous coordination agents
- **LilithListener:** Real-time message monitoring daemon *(future)*
- **SirenSnare:** WebSocket message capture system *(future)*
- **JezebelTrap:** Error handling and recovery system *(future)*
- **ZombieTrap:** Process detection and elimination *(future)*
- **SnakeCharmer:** Medusa ecosystem controller *(future)*
- **Hydra:** Multi-headed workspace coordination *(future)*

## Naming Principles

### 1. Powerful Figures from Mythology/History
- Greek/Roman goddesses (Circe, Athena, Diana)
- Historical queens (Cleopatra, Boudicca, Nefertiti)
- Mythological figures (Morgan le Fay, Siren, Hydra)

### 2. Functionality Hints in the Name
- **ZombieDust:** Creates autonomous workers
- **SirenSnare:** Lures AIs into monitoring
- **JezebelTrap:** Traps errors cunningly
- **Pandora-Box:** Contains all messages

### 3. Memorable & Professional
- Clever enough to remember easily
- Professional enough for any context
- Themed enough to maintain consistency

## Command Verbs (Medusa-themed)

- `unleash` (instead of "start") - release the snakes
- `petrify` (instead of "restart") - turn problems to stone
- `reflect` (instead of "status") - show state via mirror
- `zombify` (instead of "watch") - transform into autonomous worker
- `possess` (instead of "force") - take supernatural control
- `charm` (instead of "sync") - synchronize all serpents
